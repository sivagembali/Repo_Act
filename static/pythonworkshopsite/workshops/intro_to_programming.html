Intro to Programming

Programming is a skill to make machines do human works. Intro to programming workshop is our onboarding workshop to build that foundation and connection with the programming world. Good for someone who randomly learnt bits and pieces from different subjections or areas.

Prerequisites Pre-Evaluation What you will learn Workshop-schedule Upcoming Workshops Post-Evaluation What next?
Prerequisites

1. Basic Math Knowledge
2. Basic English
3. Basic Problem Solving Skills
4. Laptop (Wifi connectivity)

Pre-Evaluation

you will go through a pre-evaluation test before the workshop to understand your present stage of knowledge on programming. We will personalize the workshop for you accordingly.

What you will learn?
  1. Programming Concepts:
    • Commands and Usage
    • Functions
    • Top Down Design
    • Commenting
    • If/Else
    • For loop
    • While loop
    • Code Indentation
  2. Programming Approach (5 stages):
    • Stage 1: Description, Conditions, I/O, O/P, Test Cases
    • Stage 2: Breaking down into Parts
    • Stage 3: Write Pseudo Code for each part and linking code
    • Stage 4: Convert the Pseudo code into Real program
  3. Approach to learn any Programming Language:
    • Learning with analogy
    • Logic and Approach is same
